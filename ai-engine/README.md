# OmniFlow AI Engine

## Objective

The AI Engine is responsible for understanding user requests, planning tasks, selecting tools, maintaining conversation memory, and returning structured responses to the backend.

## Pipeline

User
↓
Planner
↓
Intent Classifier
↓
Task Router
↓
Tool Executor
↓
Memory Manager
↓
Response Generator
